# Lógica BASE

Este proyecto esta orientado a ser usado como base lógica. Este proyecto fue generado basándose en [Node JS](https://nodejs.org/es/) versión 14.17.0.

## Importante

En la rama `master` se encuentra el codigo que debe estar en produccion, la rama `develop` es donde van los cambios para ser probados antes de ser enviados a la rama `master`.

## Builds

Para realizar un build y probar los servicios es necesario ejecutar los siguientes comandos:

 - `npm install` Instala las dependencias necesarias y complementarias para ejecturas el proyecto. Esto creará el directorio `node_modules/`
 - `npm start`
